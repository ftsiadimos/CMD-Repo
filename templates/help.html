Command‑Library CLI Help

This page documents the available sub‑commands for repocomcli (the renamed client). Each section shows the syntax and a brief description.

1. add – Add a new command

repocomcli add command [-d description] [-t tags]

Creates a new command entry. -d is optional and provides a description. -t accepts a comma‑separated list of tags.

2. list – List commands

repocomcli list [-s search]

Displays a paginated table of commands. -s filters by text in any field. -p selects the page number; -n sets the number of rows per page.

3. show – Show a command by ID

repocomcli show id

Retrieves and prints the full details of the command with the given numeric ID.

4. config – Manage API connection settings

repocomcli config set host [--port port]

Stores the API host (and optional port) in the local configuration file.

repocomcli config get

Displays the currently stored API base URL.

5. help – Show this help page

repocomcli help

Displays this help page in the browser.
